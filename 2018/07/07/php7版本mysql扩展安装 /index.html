php7版本mysql扩展安装
2018-07-07 21:19:32
Linux 环境安装php 版本7及以上的时候，发现没有mysql扩展包，查资料后发现，在7中已被不推荐安装，如果安装需要通过以下方法
1、cd /root/
　　　2、tar zxvf mysql-230a828.tar
　　　3、cd mysql-230a828
　　　4、/usr/local/php/bin/phpize
　　　5、./configure –with-php-config=/usr/local/php/bin/php-config –with-mysql=mysqlnd
　　　6、make && make install
　　　
　　　一般支持安装包，只需要在现有安装包下phpize后生成.So文件即可
