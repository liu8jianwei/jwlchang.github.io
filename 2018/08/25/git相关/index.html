Git 2018-08-25 18:53:00
　　Git 建立与远程的关联，因为特殊原因把之前的代码删除了，在其他地方copy一份，而不是拉取远程的代码，所以需要进行如下操作
　　
1、git init 初始化一个本地仓库
2、git add .
3、Git commit - m ‘’
4、Git remote add origin {远程的仓库地址}
5、Git remote -v
　　//接下来我想在分支上进行操作，但是找不到远程分支，所以需要
　　6 git fetch
　　7 git branch -a //展示所有的分支,可看到远程所有的分子
　　8 git checkout -b {分支}
　　9 创建本地分支 git branch {}
　　10 切换分支 git checkout {}
　　11 关联远程分支 git branch –set-upstream debug origin/{}
　　
12 git pull git merge git push
